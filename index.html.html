Lenguaje de programación Html (I)
Lección #1

Para comenzar con la lección abre el block de notas o notepad y escribe lo siguiente:
< html >
< /html >
Cada una de estas cosas son llamadas "etiquetas" (tags en inglés) piensa en ellas como la manera de hablar con tu navegador o browser o mejor dicho como dar ordenes a tu browser. Hay una etiqueta para abrir y una para cerrar, para indicar la de cerrar simplemente agrega una diagonal / a la etiqueta de abrir. No todas las etiquetas tienen una etiqueta para cerrar.

Lo que le acabas de escribir indica al navegador que este es el principio de una documento HTML < html > y que este es el final del documento HTML < /html >.

Todo documento HTML necesita un par de etiquetas HEAD.

HEAD se usa para indicar los encabezados de la página Web como titulo, autor entre otros.
< html >
< head >
< /head >
< /html >

 

Lo único que nos importa poner por ahora dentro de las etiquetas HEAD es la etiqueta TITLE.
< html >
< head >
< title > titulo de la página
< /title >
< /head >
< /html >

 

Ahora, escribe un titulo a tu página, por ejemplo: Mi primera página en Internet
< html >
< head >
< title >
Mi primera página en Internet

< /title >
< /head >
< /html >

 

Ya indicado el titulo de la página a continuación se indica su contenido. Todo lo que quieras que lleve tu página Web lo debes escribir entre las etiquetas body
< html >
< head >
< title >Mi primera pagina de Internet
< /title >
< /head >
< body >
< /body >
< /html >

Por ejemplo:

< html >
< head >
< title >Mi primera página en Internet < /title >
< /head >
< body >

Mi primera página en Internet
< /body >
< /html >

Ahora almacena el documento en un disco flexible o en el disco duro en un folder de tu preferencia. Si no sabes como hacerlo a continuación te lo explicamos:

1.- Posiciona tu ratón(mouse) en la Opción Archivo (File) y da clic con el botón derecho.

2.- Selecciona Guardar como… (Save As…), en la opción Guardar como tipo (save as type) asegurate que aparezca todos los archivos (*.*)o (all files *.*), escribe el nombre del archivo con la extensión .html (los que usan windows 3.1 guárdenlo como documento1.htm) y presiona Enter.

 

Felicitaciones! Tu eres el orgulloso creador de una página Web, si la pones en un "servidor" el mundo entero podrá ver tu obra.

 

 

Para ver la página Web en tu navegador sigue las siguientes instrucciones:

1.- Posiciona tu ratón en la Opción Archivo (File) y da clic con el botón derecho.

2.- Selecciona Abrir (Open), en el recuadro, escribe la ruta y el nombre de tu página y presiona la tecla de Entrar (Enter, Return), sino, busca el folder en cual la guardaste y has doble click en el nombre de tu página para abrirla.

 

Nota: Este tutorial lo podrás aprender mejor si tienes dos ventanas de tu explorador abiertas al mismo tiempo, en una de ellas para ver este tutorial y en la otra para observar la página que ya creaste y por supuesto el Block de Notas (Notepad) para que puedas seguir añadiendo etiquetas a tu página.

 

Ahora observa la página Web que tienes en tu navegador: En la parte superior aparece el Titulo y en el área de la página aparece lo que está dentro de body.

 

¿No te parece que tu página esta un poco Vacía?. Yo creo que si, lo que acabas de crear es solo un esqueleto de una página HTML, todos los documentos de éste tipo requieren de que tengas por lo menos estas etiquetas en el.

 

 

Bueno, es todo por esta lección, nos vemos en la lección número 2
