Sobre HTML5 y CSS3 - El Futuro de la Programación Web

nuevo cambio

Este sitio web esta diseñada para servir como un primer curso universitario sobre los estándares web en informática. Además de los conceptos básicos de los estándares web, también se presentan las nuevas características de HTML5 y CSS3.

De ninguna manera esta página web debe considerarse como un curso completo de autoaprendizaje. El objetivo de esta página web es a ayudar a enseñar un curso en el aula. Sin embargo, es posible usarla como una guía de autoaprendizaje, pero con un gran esfuerzo.

Los temas cubiertos incluyen: desarrollo web actual, estandares web, World Wide Web Consortium (W3C), lenguajes de marcado, HTML5, y CSS3.

Deberias esperar gastar aproximadamente 30 horas en este curso.

Requisitos

Deberias de tener algo de experiencia en HTML (HTML 4 o XHTML 1.0) y CSS (CSS1 o CSS2).

Sobre este curso

Este curso fue impartido en la Al-Farabi Kazakh University (Kazakhstan) en Diciembre de 2011.

Sobre este sitio web

Esta página web esta hecha con HTML5 y CSS3. Esta página web usas un diseño adaptativo para adaptar la disposición al entorno de visión del usuario. Puedes probarlo cambiando el tamaño de la ventana del navegador.

Este sitio web es una traducción al castellano del sitio web HTML5 and CSS3: The Future of Web Programming realizada por Carlos Carrillo Boj.
